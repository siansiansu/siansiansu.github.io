- Sort

```cpp
class Solution {
public:
    int majorityElement(vector<int>& nums)
    {
        sort(nums.begin(), nums.end());
        return nums[nums.size() / 2];
    }
};
```

- T: $O(\log N)$
- S: $O(1)$
