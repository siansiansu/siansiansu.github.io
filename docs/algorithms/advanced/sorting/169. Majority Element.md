:::spoiler Solution - Sort
```cpp=
class Solution {
public:
    int majorityElement(vector<int>& nums)
    {
        sort(nums.begin(), nums.end());
        return nums[nums.size() / 2];
    }
};
```
- 時間複雜度：$O(\log N)$
- 空間複雜度：$O(1)$
