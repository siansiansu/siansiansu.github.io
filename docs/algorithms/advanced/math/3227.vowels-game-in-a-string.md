# [3227\. Vowels Game in a String](https://leetcode.com/problems/vowels-game-in-a-string/)

```cpp
class Solution {
public:
    bool doesAliceWin(string s)
    {
        for (int i = 0; i < s.size(); ++i)
        {
            if (s[i] == 'a' || s[i] == 'e' || s[i] == 'i' || s[i] == 'o' || s[i] == 'u') return true;
        }
        return false;
    }
};
```

- T: $O(n)$
- S: $O(1)$
