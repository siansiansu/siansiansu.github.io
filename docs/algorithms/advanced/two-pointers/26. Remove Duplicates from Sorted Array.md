:::spoiler Solution
```cpp=
class Solution {
public:
    int removeDuplicates(vector<int>& nums)
    {
        int cnt = 1;
        for (int i = 1; i < nums.size(); ++i)
        {
            if (nums[i - 1] != nums[i])
            {
                nums[cnt++] = nums[i];
            }
        }
        return cnt;
    }
};
```
- 時間複雜度：$O(N)$
- 空間複雜度：$O(1)$
