:::spoiler Solution
```cpp=
class Solution {
public:
    int removeElement(vector<int>& nums, int val)
    {
        int cnt = 0;
        for (int i = 0; i < nums.size(); ++i)
        {
            if (nums[i] != val)
            {
                nums[cnt++] = nums[i];
            }
        }
        return cnt;
    }
};
```
- 時間複雜度：$O(N)$
- 空間複雜度：$O(1)$
