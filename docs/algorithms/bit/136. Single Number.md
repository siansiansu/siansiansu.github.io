# [136\. Single Number](https://leetcode.com/problems/single-number/)

- 找出數列中的不重複元素。
- XOR

:::spoiler Solution
```cpp=
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int res = 0;
        for (auto& num : nums)
            res ^= num;
        return res;
    }
};
```
- 時間複雜度：$O(N)$
- 空間複雜度：$O(1)$
:::
