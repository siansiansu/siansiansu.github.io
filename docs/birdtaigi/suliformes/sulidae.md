## [紅腳鰹鳥 アカアシカツオドリ Red-footed Booby](https://ebird.org/species/refboo)

:::info[台語名]

- 🎯 **紅跤海雞母 âng-kha-hái-ke-bó**
- ✳️ 海雞母 hái-ke-bó

:::

:::tip[註解]

- Red-footed -> 紅跤 pe̍h-kha
- 海雞母 hái-ke-bó - 《Hêng góa Tâi-oân Chiáu-á Miâ（還我台灣鳥á名）》

:::
